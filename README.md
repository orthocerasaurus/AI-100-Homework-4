# AI-100-Homework-4

## Overview
This project demonstrates how to interact with a Large Language Model (LLM) using Python code and an API. The goal of the assignment is to practice sending prompts to an LLM programmatically and analyzing how different inputs affect the outputs.

For this assignment, I used the Hugging Face Inference API with an open-source model.

---

## Model Used
- Hugging Face model: `google/flan-t5-base`
- API: Hugging Face Inference API (free tier)
